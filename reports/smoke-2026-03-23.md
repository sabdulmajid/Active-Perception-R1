## Interpretation

- There is clear headroom from better perception policy: `oracle_crop` beats baseline on both models.
- Current prompt-only active loop underperforms baseline because these models rarely emit valid `zoom_roi` calls (`~0–4%` usage).
- This is a concrete, measured gap: the reward scaffold is present, but tool-use behavior is not yet aligned strongly enough to realize gains.
